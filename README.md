# lecture-annotation

AI-generated lecture notes website for long-form course videos.

This repo publishes annotated course notes built from:
- video subtitles / transcripts
- extracted key frames
- LLM explanations of formulas and concepts

Current site content includes courses such as:
- 量子场论（贾宇）
- 李群李代数（梁灿彬）

## Site

Published with MkDocs + GitHub Pages.

## Repository Structure

```text
lectures/
  yt-VIDEO_ID/
    lecture_notes.md
    frames/
docs/
  index.md                # auto-generated homepage
scripts/
  prepare_docs.py         # copies lectures into docs/, preprocesses math, regenerates nav
mkdocs.yml
```

## Workflow

1. Generate lecture notes in the upstream pipeline repo (`stream-polyglot/lecture_annotator`)
2. Copy output into this repo under `lectures/yt-VIDEO_ID/`
3. Run:
   ```bash
   python3 scripts/prepare_docs.py
   ```
4. Commit and push to `main`
5. GitHub Actions deploys the site automatically

## Add a New Lecture

```bash
mkdir -p lectures/yt-VIDEO_ID/
cp /path/to/lecture_notes.md lectures/yt-VIDEO_ID/
cp -r /path/to/frames lectures/yt-VIDEO_ID/
python3 scripts/prepare_docs.py
git add -A
git commit -m "feat: add <lecture title>"
git push origin main
```

## Notes

- `docs/index.md` and the `nav` section of `mkdocs.yml` are generated by `scripts/prepare_docs.py`
- Do not hand-edit generated homepage/nav unless you also update the generator
- Math in lecture notes is preprocessed for reliable MkDocs/MathJax rendering
- Large frame assets are tracked with Git LFS
